## Summary on Chapter One in the Little Go Programming Book.

1. Go is a compiled, statically typed language with a C-like syntax and garbage collection.

2. Compilation speeds is one of the major designs of Go,it's best for large projects ,gives quick feedback, tends to run faster and the executable can be run without additional dependencies. 

3. In Go, the import keyword are used to declare the packages that are used by the code in the file.
